Posted on 2023-07-20 Edited on 2023-07-30 In N3

N3文法形式の整理

いろいろな働きをする助詞

助詞 意味 例文
こそ 表示唯一、排他 今度こそ優勝したい
でも 用于提示极端事例，暗示其他事例自然不在话下 そんなことは子どもでも知っている
表示某个建议、想法或请求 お茶でも飲みましょうか
も 表示对否定的强调 一日も休まないで学校に通った
用于提示极端事例，暗示其他事例与此相同 足が痛くて立つこともできない
さえ 用于提示极端事例，表示说话人非常意外的语感 冷蔵庫には卵さえ入っていない
用于提示最必要或最低限度的事项 雨さえ降らなければ、花火ができる
まで 表示涵盖范围之广出乎人的意料之外 会ったことがない人にまで年賀状を出した
ぐらい・くらい 用于提示程度最低的事项 簡単なあいさつぐらいなら日本語で言える
用于提示程度相仿的事项或事例 この車はわたしにも買えるくらいの値段だ
など・なんか 用于委婉提出某项建议暗示尚有其他选择 連休にどこかに行こうよ。ハワイなんかどう
表示轻蔑或谦虚 ダイエットなどしたくない
だけ 表示限定 私は動物が好きだが、へびだけは嫌だ
表示最大范围 好きなだけ食べてもいいよ

助詞のような働きをする言葉

〜について

例文

説明　「〜の内容・〜に関係があることを…する。」…は「思う・考える・話す・聞く・調べる・書く・説明する」などの動詞の文。

〜に対して・〜に対する

例文　

説明　「〜に向けて…する・…という態度を示す。」…は〜への行為や態度などを表す文。（求める・文句を言う・感謝する・厳しい・親切）

〜によって

例文

説明　…は受身文。〜はその行為をする人。

〜によって…・〜によっては…

〜によって…・〜による

〜にとって

例文

説明　「〜の立場から考えると…だ。」

〜として

例文

説明　「〜という立場、資格、役割、名目で…。」

「こと・の」の使い方

「こと」だけを使う場合

(a) 「〜は…ことだ」の文で、〜の内容を…で示すとき

  • サッカーというスポーツの特徴は、基本的に手を使ってはいけないことだ

(b) 「こと」を使う文法形式

  1. 〜ことがある　→ 「たまに〜の場合がある」
  • 妻はぼくか話しかけても返事をしないことがある。
  1. 〜ことはない　→ 「〜する必要はない・〜しなくてもいい」
  • 少し熱があるが、ただの風邪だろう。すぐに病院に行くことはない

〜ということだ・〜とのことだ

〜ないことはない

〜ことは〜が、…

〜こと

〜ことにする・〜ことにしている

「の」だけを使う場合

(a) 感覚でとらえた音や光景などを言うとき（「見る・見える・眺める・聞こえる・感じる」）

  • あの日、家が大きく揺れるのを感じた。

(b) 「〜のに（は）…」の文で、〜という目的についての評価（便利だ・役に立つ）を…で言うとき

  • 短時間で食事をするのにはファストフードがやはり便利だ。

(c) 「〜のは…だ」の文で、強調したい情報を…で示すとき

  • 遅く帰ったのは残業があったからだ

(d) 「の」を使う文法形式

  1. 〜というのは…だ　→ 「〜の意味は…だ」

  2. ~のではないか・〜のではないだろうか　→ 「〜と思う」

「よう」のいろいろな使い方

「よう」を使った文法形式は「似ていることを表すもの」と「期待すること・要求することを表すもの」の二つの意味に分けられます。

〜（かの）ようだ・〜のようだ・〜（かの）ように…・〜のように…

例文

バケツをひっくり返したような雨だった
今日は暖かくて、まるで春が来たかのようだ

説明　よく似ているものに例えて言う

接続
名の＋ようだ・ように
名だである＋かのような・かのように
動普通形＋（かの）ようだ・（かの）ように

〜ように…

例文A

人間のように、植物にも栄養が必要だ。
母親が明るい人だったように、その娘たちも性格が明るい。
私たちはあなたが想像しているような関係ではありませんよ。

説明A　大体同じであることを表す

接続A　名の・普通形（ナ形だな／である・名だな／である）＋ように

例文B

日本語のように、使う文字が三種類もある言語は珍しい　

説明B　例を示す

接続B　名の＋ように

〜ように…

例文A

前に話したように、来週は私は日本にいません。
今朝の新聞に書いてあったように、今年は米の出来がいいらしい。

説明A　…の内容がすでに知られていると前置きする

接続A　動辞書形／た形／ている＋ように

例文B

よく眠れるようにワインを少し飲んだ。
赤ん坊が目を覚まさないようにテレビの音を小さくした。

説明B 期待することを表す

接続B　動辞書形／ない形＋ように

〜ように…・〜ようにと…・〜よう…

例文

雑誌を買ってくるように頼まれた。
電車の中では携帯電話で話さないようにと注意された。
今週中にご返事くださいますよう、お願い申し上げます。

説明　要求することを表す

接続　動辞書形／ない形＋ように・ようにと・よう

「わけ」のいろいろな使い方

〜わけだ・〜というわけだ

例文　

ここから東京駅まで1時間半が。じゃ、今から出れば9時には着くわけだ
放課に空いている日がなかったから、その日に会うことにしたわけです

説明　「事情から考えると、当然〜という結論になる。」

接続
普通形（ナ形だな／である・名だの／な／である）＋わけだ
普通形（ナ形（だ）・名（だ））＋というわけだ

〜わけにはいかない

〜わけにはいかない・〜わけにもいかない

〜ないわけにはいかない

例文

このCD、ずっと持っていたいけど、図書館のだから返さないわけにはいかない　

説明　「心理的事情があるので、〜しなければならない。」

接続　動ない形＋わけにはいかない

〜わけがない

〜わけではない・〜というわけではない

「ばかり」のいろいろな使い方

〜ばかり…

例文

ありがとうございます。いつもいただくばかりで、お返しもできなくてすみません。
子供はただ泣いているばかりで、何があったのか分からなかった。

説明　「いつも〜だけで、ほかのもの・ことはない。」よくないと思っていることを言う場合が多い。

接続　名（＋助詞）・動辞書形／ている＋ばかり

〜てばかりいる

例文

祖父は最近怒ってばかりいる。
二十歳のころは遊んでばかりいた。勉強しなかったことを今は残念に思っている。

説明　「ほかのことはしないで、よく〜する。」非難の気持ちで言う。

接続　動て形＋ばかりいる

〜ばかりでなく…

例文

彼は町を案内してくれたばかりでなく、この地方の料理もごちそうしてくれた。

説明　「〜だけでなくて、そのほかにも…。」

接続　名（）・普通形（ナ形だな／である・名だである）＋ばかりでなく

〜ばかりだ

例文A

外国語はいつも使っていなければ忘れていくばかりだ

説明A　「〜という一方方向に変化が進んでいく。」〜は変化を表す動詞（弱くなる・減る）。よくない方向の場合っが多い。

接続A　動辞書形＋ばかりだ

例文B

食事の準備が終わって、もう食べるばかりになっている。
旅行の準備はできました。もう出発するばかりです

説明B　「準備が終わって、後はただ〜するだけの状態だ。」

接続B 動辞書形＋ばかりだ

〜たばかりだ

例文

さっきご飯を食べたばかりなのに、もうおなかがすいてしまった。
先月結婚したばかりなので、まだ新しい生活に慣れていない。

説明　「〜したすぐ後だ。」

接続　動た形＋ばかりだ

「する・なる」の整理

する なる
〜にする・〜くする
→状態を変える
私は大きいケーキを半分にした
電気を消して、部屋を暗くしたください 		〜になる・〜くなる
→状態が変わる
大きいケーキが半分になった
電気を消したので、部屋が暗くなった
〜にする・〜ことにする
→決める
旅行の出発日は8月30日にしよう
連休にハワイに行くことにした 		〜になる・〜ことになる
→決まる
旅行の出発日は八月30日にになった
〜にしている・〜ことにしている
→決めたことを続けている
昼食はいつもパンにしています
寝る前に必ず日記を書くことにしている 		〜になっている・〜ことになっている
→決まったことが続いている
毎年、花見の会場は桜公園になっています
この会社では制服を着ることになっている
学生は車で通学できないことになっている
〜ようにする（A）
→ある目的ために変化を起こす
机の位置を変えて、仕事中でも外の景色が見えるようにしよう
ドアに穴を空けて、猫が通れるようにした 		〜ようになる
→変化が起きる
机の位置を変えてので、仕事中でも外の景色が見えるようになった
親が楽しそう家事をしていれば、子どもも進んで手伝うようになる
〜ようになっている
→ある目的のためにそう作られている
トイレに入ると、電気が自動でつくようになっている
〜ようにする・〜ようにしている
暑い日には十分水分を取るようにしましょう

「たら・ば・と・なら」の特別な使い方

〜と…た・〜たら…た

例文A

玄関を出ると、そこに大男が数人いた
窓の外を見たら、真っ白な雪景色だった

説明A　「〜の動作をすることで、…という事実に気がついた。」…は前から続いている状態を表す文で、少し意外感がある内容。「…ていた」の形がよく使われる。〜と…の主語は違う。

接続A　動辞書＋と　動たら

例文B

夜12時ごろテレビを見ていると、アメリカの友達から電話がかかってきた
庭の掃除をしていたら、突然大雨が降り出した

説明B　「〜の動作をしているときに、偶然…起こった。」〜は「〜ていると、〜ていたら」の形が多い。…は動詞の文で、話者の意志を含まない、意外感がある内容。

接続B　動ている＋と　動たら

例文C

久しぶりに高校の先生に手紙を出すと、すぐに返事が来た。
料理にちょっとお酒を入れてみたら、いい味になった。

説明C　「〜の動作がきっかけになって、…が起こった。」

接続C　動辞書＋と　動たら

〜と…た

例文

その男は私の顔を見ると、すぐに逃げていってしまった
ゆき子は飛行機から降りると、どこかに電話をかけた

説明　「〜の動作の後、続けてすぐ…した。」〜と…の主語は同じ。

接続　動辞書＋と

〜も…ば〜も…・〜も…なら〜も…

例文

人生には楽しい時もあればつらい時もある

説明　「〜も…し、〜も…・〜も〜も両方…・」「も」を２回使う

接続

動詞や名詞の意味を広げる文法形式

動詞や名詞に別の言葉をつけると、その動詞や名詞の意味を広げることができます。

ある動作のどの段階がを表す

文法形式 意味 例文
〜かける
〜かけだ 		全部〜し終わっていない 彼女は何か言いかけて、黙ってしまった。 母は読みかけの本をおいて、買い物に出かけた。
〜きる 全部／完全に〜する 40キロの長い距離を走りきった
こんなにたくさん、一人では食べきれない
〜通す 最後まで〜し続ける 自分で決めたことは最後までやり通そう
彼は何を聞かれても黙り通した
〜出す （突然・急に）〜し始める 彼は私の顔を見ると、突然笑い出した
電池を入れ替えたら、おもちゃの車が急に動き出した

特徴・状態・様子を表す

文法形式 意味 例文
〜やすい 簡単に〜できる
簡単に〜してしまう 		薄くて破れやすい紙だから気をつけて
〜にくい 簡単に〜できない
簡単には〜しない 		この肉は硬くて食べにくい
〜づらい 〜するのが難しくて困る 個人的なことなので職場の人には頼みづらい
〜すぎる
〜すぎだ 		適切な程度を超えて〜だ・〜する おいしかったので、食べすぎてしまった
〜がちだ 〜の状態になる傾向がある 雨のため工事は遅れがちだ
〜らしい 〜のイメージと同じだ 今日は暖かくて春らしい一日だった
〜っぽい 〜のような要素・性質がある 彼女はわがままばかり言って、子どもっぽい人だ
〜のようだ
〜みたいだ 		〜によく似ている ここから見える景色は絵のようだ
〜だらけだ 〜（いやなもの）がたくさんある 返ってきたテストは間違いだらけだった
