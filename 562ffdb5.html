文の組み立て

文の組み立てー1 引用

ある人が言ったこと、自分が思ったことの内容を一文の中に入れるには、つぎのような方法があります。

「言う・思う・感じる」など

普通形＋と…

  • わたしは思います。＋「リンさんはきっと合格するでしょう」
    →わたしはリンさんはきっと合格するだろうと思います。

「頼む・注意する・言う」など

「辞書形・ない形」＋ように（と）
「て形・ない形＋で」＋ほしい＋と
命令形・禁止形＋と

  • 医者はカンさんに勧めた。＋「ときどき運動をしたほうがいいです」
    →　医者はカンさんにときどき運動するように勧めた。
  • わたしは先生に頼んだ。＋「もう一度説明してくださいませんか」
    → わたしは先生にもう一度説明してほしいと頼んだ。

「聞く・わからない・調べる」など

普通形＋かどうか
疑問詞＋〜か

  • わたしはカンさんに聞いた。＋「あした会えるでしょうか」。
    → わたしはカンさんにあした会えるかどうか（を）聞いた。
  • わたしはわからない。＋「どうすれば日本語が上手になるでしょうか」
    → わたしはどうすれば日本語が上手になるか（が）わからない。

どうして／なぜ〜かというと、…からだ

  • 「どうしてこの店は人気がありますか」＋安くておいしいからです。
    → どうしてこの店は人気があるかというと、安くておいしいからです。

文の組み立てー2　名詞の説明

「普通形（ナ形だな・名だの）」＋名

  • そのバスは東京駅に行く　→ 東京駅に行くバス
  • そのニュースを聞いてショックを受けた　→ ショックを受けたニュース
  • にぎやかな声で子どもたちが話している　→ 子どもたちが話しているにぎやかな声

「助詞＋の」＋名

  • そのバスは駅まで行く　→ 駅までのバス
  • 母から手紙が来た　→ 母からの手紙
  • 友達にプレゼントをあげる　→ 友達へのプレゼント

「助詞のような働きをする言葉」＋の＋名

  • 生命について本を書いた　→ 生命についての本

「助詞のような働きをする言葉の名詞につく形」＋名

  • そのバスは市の運営によって走っている　→ 市の運営によるバス

「状態や様子を表す言葉（ばかり・とおり・まま・はず）」＋の＋名

  • そのバスは修理したばかりだ　→ 修理したばかりのバス

文の組み立てー3　「〜という・〜といった」

「〜という…・〜といった…」の「〜」は「…名詞」の名前・内容・例を示します。はっきり言うのを避けるときは「という・といった」の後に「ような」をつけます。

名前を紹介する（〜という…）

  • 私は松下という者です
  • これ、珍しいですね。何という果物ですか

内容を示す（〜という…）

  • 鈴木さんが会社を辞めるといううわさを聞いた
  • 「ワンワン」という犬の鳴き声が聞こえます
  • 「三日坊主」はすぐにあきてしまうというような意味だと思います

知識・事実などを示す（〜ということ）

  • かぜの予防には手洗いがいいということを知っていましたか
  • まき子さんのお母さんは有名な女優だということを初めて聞いた
  • 兄はこの前会ったとき、今月は忙しいというようなことを言っていた

よく知らない言葉を示す（〜というの）

  • コンピューター用語で、アップデートというのは何ですか

いくつかの例を示す（〜といった…）

  • 私は数学、物理、化学といった理科系の科目が好きだ
  • 彼女は黒とか灰色といったような暗い色の服が似合う

文の組み立てー4　決まった形

組み合わせが決めって文法形式があります、決まった形として覚えましょう。

〜から…にかけて　→ 〜から…の間

  • 年末から年始にかけて私はとても忙しい
  • 明日は関東地方から東北地方にかけて雨が降るでしょう

〜を…として・〜を…に　→　〜を…考えて（認めて）

  • 入院をきっかけとして私は健康に注意するようになった
  • この小説家は家族をテーマとして小説をたくさん書いている

〜はもちろん…も　→ 〜は当然だが、…も同じように

  • このゲームは子どもはもちろん大人も楽しめる
  • 夫は国内の山々はもちろん外国の山にもあちこち登っている

（〜ば）〜ほど…・（〜なら）〜ほど…

  • 考えれば考えるほどわからなくなってきた

〜くらい…はない・

〜さえ〜ば…・〜さえ〜なら…

〜ことは〜が、…
